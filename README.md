\# Sign Language Project



This project converts sign language gestures into text and speech using Python.



\## Technologies Used

\- Python

\- Streamlit

\- OpenCV

\- TensorFlow

\- pyttsx3



\## Features

\- Capture hand gesture using camera

\- Convert gesture to text

\- Convert text to speech



\## Installation



Install dependencies:



py -3.11 -m pip install -r requirements.txt



\## Run the Application



py -3.11 -m streamlit run app.py
